# EstateVision Landing Page

Unified Asset Intelligence and Predictive Risk Monitoring for High-Security Environments.

## Overview

EstateVision provides complete situational awareness by ingesting and normalizing thousands of fragmented data sources into a single, cohesive ontology. This landing page demonstrates the platform's core capabilities, including:

- **Real-Time Physical Risk Monitoring**
- **Infrastructure Vulnerability Scoring**
- **Siloed Intelligence Integration**
- **Predictive Asset Intelligence**

## Technical Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Visualization**: React Three Fiber / Drei
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
